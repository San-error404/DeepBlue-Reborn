# DeepBlue-Reborn (Python Chess Engine)

A modular open-source chess engine written in Python, inspired by the architecture of Deep Blue:
- Alpha-beta search
- Quiescence search
- Transposition tables
- Move ordering
- UCI protocol support
- Modular structure for future optimization (C++ core possible)

## Install

```bash
python3 -m pip install --user -r requirements.txt
